🔧 開發指引

標籤設定方式

  1. Metadata 設定 - 在產品 metadata 中添加 promotions 欄位
  2. Tags 設定 - 在產品 tags 中添加關鍵字（new、hot、limited 等）
  3. 促銷系統 - 透過 Medusa 促銷功能建立折扣規則

買X送Y 設定範例

Tags 方式： buy-x-get-y, buyxgety, 買送, 贈品, 買1送1, 買2送1, 買3送1

促銷代碼： BUY2GET1, 買2送1, GIFTPROMO 等

Metadata 方式： { "promotions": { "buyXGetY": "送好禮" } }
