內文系統展示 Body Text System

響應式內文樣式 - 這是主要的內文樣式，在手機上顯示 16px，在桌面上顯示 18px。 使用系統字體確保最佳的跨平台顯示效果。支援中英文混排，字距經過優化。 Responsive body text with optimal readability across all devices and languages.

標準內文樣式 - Standard body text with 16px size, optimized line-height and letter-spacing. 使用現代字型堆疊確保在所有設備上都有良好的顯示效果。

大號內文樣式 - Large body text at 18px for better readability in important content areas. 適用於重要內容區域，提供更好的閱讀體驗。

小號內文樣式 - Small body text at 14px for secondary information, captions, and metadata. 用於次要資訊、圖片說明和中繼資料。

代碼字體 Monospace Font

/* CSS 變數定義 */
:root {
  --font-base: -apple-system, BlinkMacSystemFont, 'Segoe UI Variable';
  --font-heading: 'SF Pro Display', -apple-system, BlinkMacSystemFont;
  --font-mono: 'SF Mono', 'Monaco', 'Menlo', 'Consolas';
}

// JavaScript 範例
const fontStack = {
  base: 'var(--font-base)',
  heading: 'var(--font-heading)',
  mono: 'var(--font-mono)'
};

性能優化特色 Performance Features

  • 零網路請求：使用系統內建字體，載入速度最快
  • 避免 FOIT：設定 font-display: swap 避免文字閃爍
  • 原生體驗：使用者熟悉的系統字體，提升可讀性
  • 完整 Unicode：系統字體支援最完整的字符集
  • 自動優化：字距、連字、上下文替代自動啟用
  • 響應式設計：根據螢幕大小自動調整字體大小
  • 無障礙友好：遵循 WCAG 指南的對比度和可讀性
