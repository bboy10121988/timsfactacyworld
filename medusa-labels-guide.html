⚙️ 三種設定方式

🔧 方式一：產品 Metadata

在 Medusa Admin 的產品編輯頁面中設定

操作步驟：

  1. 登入 Medusa Admin
  2. 進入 Products → 選擇產品
  3. 找到 "Metadata" 區域
  4. 新增以下 key-value：
# 折扣標籤 discount: "8折" # 買X送Y buy_x_get_y: "送禮品" buyXGetY: "買2送1" # 狀態標籤 is_new: true is_hot: true is_limited: true is_bestseller: true is_featured: true is_exclusive: true # 特殊活動 special_event: "聖誕特價" campaign: "週年慶" bundle: "超值組合" flash_sale: true clearance: true # 促銷相關 promotion: "限時優惠" promotion_type: "special-event" discount_code_available: true

🏷️ 方式二：產品標籤 (Tags)

在產品的 Tags 欄位中新增標籤

操作步驟：

  1. 編輯產品頁面
  2. 找到 "Tags" 欄位
  3. 新增以下標籤：
# 中文標籤 新品, 熱銷, 限量, 暢銷, 精選, 獨家 限時搶購, 清倉, 組合優惠 # 英文標籤 new, hot, limited, bestseller, featured flash-sale, clearance, bundle, exclusive # 買送相關 買送, 贈品, 買1送1, 買2送1, 買3送1 buy-x-get-y, buyxgety

🎯 方式三：Medusa 促銷系統

使用 Medusa 的促銷模組創建促銷活動

操作步驟：

  1. 進入 Medusa Admin → Promotions
  2. Create New Promotion
  3. 設定促銷類型和條件
  4. 系統會自動生成對應標籤
# 促銷代碼範例 PERCENTAGE10 → 自動顯示折扣標籤 PRODUCT-50OFF → 商品折扣標籤 ORDER-100OFF → 訂單折扣標籤 BUY2GET1 → 買X送Y標籤 FLASH-SALE → 限時搶購標籤 # 系統自動識別關鍵字 包含 "buy", "get" → 買X送Y標籤 包含 "flash", "sale" → 限時標籤 包含 "order" → 訂單優惠標籤 包含 "product" → 商品優惠標籤

🎨 標籤自動顯示邏輯

⚡ 自動標籤（無需設定）

  • 售完/預訂： 系統根據庫存狀態自動顯示
  • 自動折扣： 系統計算原價vs現價自動顯示折扣
  • 促銷活動： 根據 Medusa 促銷模組自動顯示

🔄 標籤優先級規則

  • 數字越小優先級越高
  • 售完標籤永遠最優先顯示
  • 折扣相關標籤優先於狀態標籤
  • 同類型標籤不會重複顯示
  • 現在支援無限數量標籤同時顯示

🎯 常見設定範例

📱 新品上市

# Metadata 設定 is_new: true special_event: "新品首發" # Tags 設定 新品, new, 首發 # 顯示結果 🏷️ 新品首發 🏷️ 新品

🔥 熱銷促銷

# Metadata 設定 is_hot: true discount: "7折" buy_x_get_y: "買2送1" # 顯示結果 🏷️ 7折 🏷️ 買2送1 🏷️ 熱銷

⚡ 限時清倉

# Metadata 設定 flash_sale: true clearance: true discount: "5折" # 顯示結果 🏷️ 限時搶購 🏷️ 5折 🏷️ 清倉

🎁 組合優惠

# Metadata 設定 bundle: "超值組合" buy_x_get_y: "送配件" is_featured: true # 顯示結果 🏷️ 超值組合 🏷️ 送配件 🏷️ 精選

💡 小貼士

  • 建議每個產品不要設定過多標籤（3-5個為佳）
  • 優先使用高優先級的促銷標籤（折扣、限時等）
  • 買X送Y標籤建議只顯示「送Y」部分
  • 可以同時使用多種設定方式，系統會自動去重
  • 標籤會根據優先級自動排序顯示
