🔍 測試說明

✓
inline-block 顯示模式：標籤使用 inline-block 確保寬度跟隨文字內容
✓
w-auto 自動寬度：移除固定寬度限制，讓容器自適應文字
✓
whitespace-nowrap 防止換行：確保文字在單行顯示
✓
flex-col gap-1 垂直排列：標籤垂直堆疊，每個標籤寬度獨立

🎯 預期效果

  • • 短標籤（如「9折」、「新品」）應該緊貼文字，沒有多餘空白
  • • 長標籤（如「滿額-NT$200」）應該完整顯示，不被截斷
  • • 每個標籤寬度不同，完全根據文字內容調整
  • • 垂直堆疊時，不同長度標籤左對齊，右側自然結束

🔧 技術實現

修改的 CSS 類別：

inline-block bg-stone-800/90 text-white px-2 py-1 shadow-sm text-xs font-medium tracking-widest uppercase whitespace-nowrap w-auto

關鍵屬性：

  • inline-block - 讓標籤可以設定寬高，同時保持行內特性
  • w-auto - 寬度自動適應內容
  • whitespace-nowrap - 防止文字換行
